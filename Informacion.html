Formulario Empresa

Este formulario gestiona la información relacionada con las empresas que participan en el proceso de ventas de productos. Permite registrar, visualizar, editar y eliminar datos de la empresa. Las funciones y operaciones que se pueden realizar son las siguientes:

1.- Agregar empresa: permite registrar una nueva empresa en el sistema proporcinando todos los atributos mencionados anteriormente.

2.- Visualizar informacion de la empresa: muestra una lista de las empresas registradas en el sistema.

3.- Editar empresa: modifica la información existente de una empresa.

4.- Eliminar: elimina una empresa del sistema en caso de que ya no esté disponible.
Formulario Factura

El formulario de factura tiene las siguientes funcionalidades:

1.- Generar factura: permite crear una nueva factura para una nueva venta, solicitando número de factura, fecha de factura, código de cliente, monto de factura, sucursal y tipo moneda.

2.- Cargar facturas: muestra en una tabla la lista de facturas existentes en el sistema. proporciona toda la información sobre la factura.

3- Eliminar factura: La API procesa la solicitud y realiza la eliminación de la factura del sistema, lo que implica eliminarla de la base de datos.

4.- Actualizar factura: permitir la modificación de una factura existente en el sistema. Al actualizar una factura, se pueden modificar los detalles de la venta

Formulario Cliente

Este formulario almacena información relevante sobre los clientes, como número de cliente, nombres, apellidos, rtn, fecha de afiliación, saldo actual y numero de teléfono. Con Este formulario se pueden realizar las siguientes operaciones:

1.- Registrar nuevos cliente: añade nuevos clientes a la base de datos.

2.- Actualizar información del cliente: permite modificar la información y los datos esten actualizados.

3.- Consultar información del cliente: la API proporciona datos del cliente almacenados en la base de datos.

4.- Eliminar clientes: elimina información de la base de datos.
Formulario Producto

Este formulario gestiona la información de los productos que se ofrecen en la plataforma, contiene campos como número de factura, id del producto, nombre, fecha de vencimiento, precio, cantidad y comentarios que permiten registrar, actualizar, consultar y eliminar información de los productos. Con este formulario, se pueden realizar las siguientes operaciones relacionadas con la gestión de productos:

1.- Registro de nuevos productos: permite añadir nuevos productos a la base de datos.

2.- Actualización de información del producto: se pueden modificar la información de los productos existentes, como cambiar el precio, actualizar la descripción.

3.- Eliminación de productos: elimina el producto de la base de datos.

4.- Consulta de productos: proporciona datos de los productos almacenados en la base de datos.
